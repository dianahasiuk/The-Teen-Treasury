# The Teen Treasury

**Фінанси з відчуттям стилю. Для підлітків, які знають, що гроші — це не страшно.**

---

Навчитися керувати кишеньковими грошима та маленькими доходами здається складно. **The Teen Treasury** робить це просто. Красиво. Інтерактивно. Стильно.  

- Відстежуйте доходи та витрати.  
- Плануйте накопичення на цілі.  
- Слідкуйте за прогресом у барвистій “скарбничці”.  
- Отримуйте мотиваційні значки та короткі поради, які перетворюють фінансову грамотність на гру.  

---

Цей додаток не про цифри. Він про контроль, маленькі перемоги і розвиток корисних звичок. Навіть найменші кроки до вашої “скарбнички” формують навички, які залишаються назавжди.
